CSS SELECTORS

Element Selector

The Element selector is uesd to style all the elements of a particular type at once.

ID Selector

The ID selector is used to style only a single element of a particular type.

Class Selector

The Class Selector is used when we want to style more than one element.

Grouping Selector

The Grouping selector help us to same style to multiple sections of elements.

comma is used to seperate them,

Pseudo Class selector

Selects the elements that are in specific state.
